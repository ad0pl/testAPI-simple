AliasRecordItem

  • name : String
    short name of the hostname of the object
  • view : String
    DNS view record should be resolvable, is field is case senitive.
  • canonical : String
  • change_control : String
    Record of change that authorizes this change
  • domain : String
    Domain this DNS record belongs to
  • link : String
    uri that points to this HostRecordItem
